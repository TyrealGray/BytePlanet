UE4 nav system

2016.07.18

You can set a nav if you use Nav Mesh Bounds. This Nav system can tell AI bot where it can go or not

First Open UE4 editor, find the Nav Mesh Bounds option

Drag it on the map

Then we use it to cover all the map area and click keyboard key p to see the nav

After this, we create a Character, set blueprint like that. When the game begin, it will move to (100,200,300).
