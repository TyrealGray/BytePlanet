Implement UE4 LAN Network

2016.06.28

oh yeah!!! After my hard work, researching the Multiplayer Shootout example. Finally I figure out how to implement UE4 LAN Network. Here is my example, hope it would help to make you easily understand.

First of all, you need to create your own game instance,also need a create room button unless you want user types IP address. UMG system is very simple, no need description.

let's see the host game button function in blueprint:

in the ui system, this button trigger custom event "Host Game Event" in game instance:

see, it is very easy.Passing player controller to Create Session function, the second parameter is max number of player , we set "true" for allowed LAN network. When it succeed, open the WestTown map and other clients use Find Session function to find your room, then Join Session to enter your room.

As I said, if host is success to use Create Session function, then we can call Find Session to get the Result which contain the Session that allowed we to join to. Results is an array, the type is Blueprint Session Result.

Blueprint Session Result has several function:

to join the room, you just need pass the Blueprint Session Result to Join Session function:

then, you can find a event in game mode called "OnPostLogin" , this event fire when player enter the room successfully no matter host or join. Also you can manager the character here:

New Player is player controller reference, Client Post Login is my own function

Now your game is ready to go, hope you succeed.

ps: if Create Session is not working, you can go to the config folder, editing DefaultEngine.ini to add those lines:

  1. [OnlineSubsystem]
  2. DefaultPlatformService=Null
